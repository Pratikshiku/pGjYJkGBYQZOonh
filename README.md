## Tourism Website System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0e0244e7c64441afa8b2b610f3b91930/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Forum
5. Attraction Information
6. Tourism News
7. Travel Routes
8. Announcement Information
9. Company Introduction
10. Personal Center
11. Backend Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Basic Data Management (Attraction Type Management, News Type Management, Route Type Management, Announcement Type Management)
5. Forum Management
6. Attraction Information Management (Attraction Information Management, Attraction Information Comment Management)
7. Tourism News Management
8. Travel Route Management
9. Announcement Information Management
10. Company Introduction Management
11. User Management
12. Carousel Information
